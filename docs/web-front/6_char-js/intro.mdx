# Chart.js

Chart.js는 HTML5 Canvas를 기반으로 한 간단하면서도 강력한 JavaScript 차트 라이브러리입니다. 개발자들이 아름답고 반응형인 차트를 쉽게 만들 수 있도록 설계되었습니다.

## 주요 특징

Chart.js는 다음과 같은 장점을 제공합니다:

**간단한 사용법**: 몇 줄의 코드만으로 멋진 차트를 만들 수 있습니다. 복잡한 설정 없이도 바로 시작할 수 있어 초보자도 쉽게 접근할 수 있습니다.

**8가지 차트 타입**: 선형 차트, 막대 차트, 레이더 차트, 도넛 및 파이 차트, 극좌표 차트, 버블 차트, 분산형 차트 등 다양한 시각화 옵션을 제공합니다.

**반응형 디자인**: 모든 디바이스에서 완벽하게 작동하며, 화면 크기에 따라 자동으로 조정됩니다.

**애니메이션**: 부드러운 애니메이션 효과로 데이터 변화를 시각적으로 표현할 수 있습니다.

**커스터마이징**: 색상, 폰트, 레이블, 툴팁 등 모든 요소를 자유롭게 커스터마이징할 수 있습니다.

## 시작하기

Chart.js를 사용하는 가장 간단한 방법은 CDN을 통해 라이브러리를 불러오는 것입니다:

```html
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
```

또는 npm을 통해 설치할 수 있습니다:

```bash
npm install chart.js
```

## 기본 예제

다음은 간단한 선형 차트를 만드는 예제입니다:

```html
<canvas id="myChart"></canvas>

<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
<script>
  const ctx = document.getElementById('myChart');
  
  new Chart(ctx, {
    type: 'line',
    data: {
      labels: ['1월', '2월', '3월', '4월', '5월', '6월'],
      datasets: [{
        label: '월별 판매량',
        data: [12, 19, 3, 5, 2, 3],
        borderWidth: 1
      }]
    },
    options: {
      responsive: true,
      scales: {
        y: {
          beginAtZero: true
        }
      }
    }
  });
</script>
```

## 누가 사용하나요?

Chart.js는 전 세계 수많은 개발자들이 사용하고 있으며, 다음과 같은 경우에 특히 유용합니다:

- 대시보드 및 분석 도구 개발
- 데이터 시각화가 필요한 웹 애플리케이션
- 보고서 및 통계 페이지
- 실시간 데이터 모니터링 시스템
