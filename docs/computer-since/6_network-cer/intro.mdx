# 네트워크관리사 2급 자격증 소개

## 네트워크관리사란?

네트워크관리사는 **기업이나 조직의 네트워크 시스템을 안정적으로 운영하고 관리할 수 있는  
전문 인력을 양성하기 위한 국가공인 민간자격증**입니다.

인터넷과 네트워크가 우리 생활의 중심이 된 지금, 안정적인 네트워크 운영은 기업의 핵심 경쟁력이 되었습니다.  
네트워크관리사 자격증은 이러한 시대적 요구에 부응하여 네트워크 분야의 전문성을 인증합니다.

:::tip 왜 네트워크관리사 2급인가요?
2급은 네트워크 관리의 **실무 기초를 다지는 단계**로, IT 분야 취업을 준비하거나  
네트워크 관련 업무를 수행하시는 분들에게 적합한 자격증입니다.
:::

---

## 자격증 개요

### 시행 기관
- **한국정보통신자격협회(ICQA)**에서 주관
- 공식 홈페이지: [www.icqa.or.kr](https://www.icqa.or.kr/cn/page/network)

### 등급 구성
네트워크관리사는 **1급과 2급** 두 가지 등급으로 구성되어 있으며, 2급부터 응시 가능합니다.

| 등급 | 난이도 | 대상 |
|------|--------|------|
| 1급 | 상급 | 네트워크 시스템 설계 및 구축 전문가 |
| 2급 | 중급 | 네트워크 운영 및 관리 실무자 |

## 2급 시험 구성

### 필기시험

**시험 시간**: 50분  
**문항 수**: 50문항  
**문제 유형**: 객관식 4지선다형  
**만점**: 100점  
**합격 기준**: 60점 이상

#### 출제 과목 및 내용

**1. 네트워크 일반**
- 네트워크 개요 및 기본 개념
- 데이터통신 관련 기술
- 통신망 표준과 네트워크 설계
- 네트워크 구축 실무
- 고속 LAN 기술 및 광대역 통신
- 각종 통신 기술

**2. TCP/IP**
- TCP/IP 프로토콜 구조
- IP 주소 체계(IP Address)
- 서브넷 마스크(Subnet Mask)
- IP 라우팅(IP Routing)
- 패킷 분석(Packet Analysis)
- 기본 프로토콜(TCP, UDP, ICMP 등)
- 응용 프로토콜(HTTP, FTP, SMTP 등)

**3. NOS(Network Operating System)**
- Windows Server 환경
  - File/Print 서비스
  - IIS(인터넷 정보 서비스)
  - DNS(도메인 네임 시스템)
  - FTP 서버 구축
  - Active Directory
  - 사용자 관리 및 권한 설정
  - 시스템 운영 관리
- Linux Server 환경
  - Linux 설치 및 기본 설정
  - Linux 명령어
  - 서버 운영 실무

**4. 네트워크 운용기기**
- NIC(네트워크 인터페이스 카드)
- SCSI(저장장치 인터페이스)
- RAID(디스크 어레이)
- Router(라우터)
- L2/L3/L4 Switch
- VLAN(가상 랜)
- Gateway(게이트웨이)
- 전송매체(UTP, 광케이블 등)
- 최신 통신기기

### 실기시험

**시험 시간**: 80분  
**문항 구성**: 1세트(1~20번)  
**문제 유형**: 작업형, 서술형, 선택형 혼합  
**만점**: 100점  
**합격 기준**: 60점 이상

#### 출제 과목

1. LAN 구축 및 전송매체
2. 네트워크 설계 및 구축 실무
3. TCP/IP 실무 설정
4. NOS 운영 실무
5. 네트워크 운용기기 설정 및 관리

:::info 실기시험 특징
실기시험은 **실제 업무 환경을 기반**으로 한 문제가 출제됩니다. 네트워크 설정, 명령어 입력, 문제 해결 등 실무 능력을 종합적으로 평가합니다.
:::

## 응시 자격

네트워크관리사 2급은 **별도의 응시 자격 제한이 없습니다**. 학력, 경력, 연령에 관계없이 누구나 응시 가능합니다.

## 자격증의 활용

### 취업 및 실무
- IT 기업 네트워크 관리자
- 기업체 전산실 운영 인력
- 네트워크 보안 관련 업무
- 통신 서비스 제공업체
- 시스템 통합(SI) 업체

### 학점 인정
일부 대학에서 학점으로 인정받을 수 있습니다(학교별 상이).

### 가산점
공공기관 및 일부 기업 채용 시 우대 또는 가산점이 부여될 수 있습니다.

## 합격 전략

네트워크관리사 2급은 **이론과 실무를 균형 있게 학습**하는 것이 중요합니다.

**필기시험 준비**
- 네트워크 기본 개념 확실히 이해하기
- TCP/IP 프로토콜 체계 숙지
- 실무에서 자주 사용되는 명령어 암기
- 기출문제 반복 학습

**실기시험 준비**
- 실제 네트워크 환경 구축 연습
- Linux 명령어 실습
- 라우터/스위치 설정 실습
- 문제 해결 능력 키우기
