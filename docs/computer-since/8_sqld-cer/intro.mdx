# SQL 개발자(SQLD: SQL Developer)란?

## 개요

SQL 개발자(SQLD)는 **데이터베이스와 데이터 모델링에 대한 지식을 바탕으로  
정확하고 최적의 성능을 발휘하는 SQL을 작성할 수 있는 개발자**를 말합니다.

한국데이터산업진흥원에서 시행하는 공인자격증(제2022-04호)으로, 데이터베이스 기반 정보시스템 개발에 필수적인 SQL 활용 능력을 검증합니다.

:::tip 핵심 포인트
SQL 사용 능력 + 데이터 모델 이해 = 고품질 정보시스템 개발
:::

---

## 왜 취득해야 하나요?

SQLD 자격증을 취득해야 하는 이유는 다음과 같습니다:

- **필수 기술 검증**: 관계형 데이터베이스는 대부분의 정보시스템에서 사용되며, SQL은 데이터에 접근하는 유일한 언어입니다
- **전문성 향상**: SQL의 수행 원리를 깊이 있게 이해하고 제대로 구사할 수 있는 전문 지식을 습득합니다
- **경쟁력 강화**: 체계적인 SQL 학습을 통해 정보시스템의 성능과 품질을 향상시킬 수 있습니다
- **학점 인정**: 학점은행제 6학점으로 인정받을 수 있습니다

---

## SQL이란?

SQL(Structured Query Language)은 데이터베이스를 직접적으로 액세스할 수 있는 언어입니다. SQL은 다음과 같은 기능을 제공합니다:

- **데이터 정의(DDL)**: 테이블 구조를 생성, 변경, 삭제, 재명명
- **데이터 조작(DML)**: 데이터를 입력, 조회, 수정, 삭제
- **트랜잭션 제어(TCL)**: 조작한 결과를 적용하거나 취소
- **데이터 제어(DCL)**: 접근 권한을 제어

## 주요 직무

SQL 개발자가 수행하는 주요 직무는 다음과 같습니다:

### 데이터 모델의 이해 및 분석

데이터 모델은 데이터베이스 구성의 핵심 요소로, 건물의 설계도와 같은 역할을 합니다.  
엔터티, 속성, 관계, 식별자, 정규화 등 데이터 모델의 기본 지식을 바탕으로 데이터 모델을 이해하고 분석하는 작업을 수행합니다.

### SQL 이해 및 활용

SQL 문법을 이해하고 활용하여 다양한 작업을 수행합니다. DDL을 통해 테이블 구조를 관리하고,  
DML을 통해 데이터를 처리하며, 다양한 JOIN 방법과 서브쿼리를 사용하여 복잡한 데이터 관계를 표현합니다.

## 시험 구성

### 시험 정보

- **시행 횟수**: 연 4회 정기 시행
- **시험 시간**: 90분(1시간 30분)
- **문항 수**: 총 50문항(객관식)
- **응시 자격**: 제한 없음

### 출제 과목

SQLD 시험은 총 2과목으로 구성됩니다:

#### 1. 데이터 모델링의 이해 (10문항, 20점)

- **데이터 모델링의 이해**: 데이터모델의 이해, 엔터티, 속성, 관계, 식별자
- **데이터 모델과 SQL**: 정규화, 관계와 조인의 이해, 트랜잭션의 이해, Null 속성의 이해, 본질식별자 vs 인조식별자

#### 2. SQL 기본 및 활용 (40문항, 80점)

- **SQL 기본**: 관계형 데이터베이스 개요, SELECT 문, 함수, WHERE 절, GROUP BY/HAVING 절, ORDER BY 절, 조인, 표준 조인
- **SQL 활용**: 서브쿼리, 집합 연산자, 그룹 함수, 윈도우 함수, Top N 쿼리, 계층형 질의와 셀프 조인, PIVOT/UNPIVOT절, 정규 표현식
- **관리 구문**: DML, TCL, DDL, DCL

## 학습 방향

SQLD 자격증을 준비하시는 분들께 다음과 같은 학습 방향을 추천드립니다:

1. **데이터 모델링 기초**: 엔터티, 속성, 관계의 개념을 확실히 이해하세요
2. **SQL 기본 문법**: SELECT, WHERE, JOIN 등 기본 문법을 반복 연습하세요
3. **SQL 활용**: 서브쿼리, 윈도우 함수 등 고급 기능을 익히세요
4. **실습 중심**: 이론만이 아닌 실제 쿼리 작성 연습이 중요합니다
5. **과목별 균형**: 두 과목 모두 과락 기준을 넘어야 하므로 균형있게 학습하세요
