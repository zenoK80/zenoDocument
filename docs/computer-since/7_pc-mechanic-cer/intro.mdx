# PC정비사 2급 자격증 소개

## PC정비사란?

PC정비사는 **컴퓨터 하드웨어와 소프트웨어를 종합적으로 관리하고 유지보수할 수 있는 능력을 인정**하는 국가공인 민간자격증입니다. 

컴퓨터가 고장 났을 때 문제를 진단하고 수리하는 것은 물론, PC를 조립하고 네트워크를 구축하는 등  
실무에 필요한 전문 기술을 갖춘 전문가를 양성하기 위한 자격증입니다.

:::tip 왜 PC정비사 자격증이 필요할까요?
- IT 분야 취업 시 **실무 능력을 객관적으로 증명**할 수 있습니다
- 컴퓨터 수리 및 유지보수 업체 창업 시 **전문성을 인정**받을 수 있습니다
- 학교, 기업, 관공서 등에서 PC 관리 업무를 수행할 수 있는 **실무 역량**을 갖출 수 있습니다
:::

## 시험 구성

PC정비사 2급 시험은 **필기시험**과 **실기시험**으로 구성되어 있으며, 각각 독립적으로 응시할 수 있습니다.

### 필기시험

| 구분 | 내용 |
|------|------|
| **시험시간** | 50분 |
| **문제유형** | 택일형 (객관식) |
| **문항수** | 50문항 |
| **만점/합격점** | 100점 / 60점 이상 |

#### 출제 범위

**1. PC 운영체제 (Operating System)**
- 운영체제의 개요와 기본 개념
- Windows 계열 운영체제 활용
- 시스템 유틸리티 사용법

**2. PC와 주변장치 (Hardware)**
- 중앙처리장치 (CPU)
- 기억장치 (메모리, HDD, SSD)
- 입력장치 (키보드, 마우스 등)
- 출력장치 (모니터, 프린터 등)
- 기타 주변장치

**3. PC 유지보수**
- PC 부품의 조립 및 설정
- 고장 진단 방법
- 시스템 업그레이드

**4. PC 네트워크**
- 네트워크 기본 개념
- 네트워크 장비 (라우터, 스위치 등)
- 네트워크 프로토콜
- 인터넷 연결 및 설정

### 실기시험 (중급 실무)

| 구분 | 내용 |
|------|------|
| **시험시간** | 80분 |
| **문제유형** | 작업형 / 서술형 / 선택형 |
| **세트수** | 1 SET (1-20번) |
| **만점/합격점** | 100점 / 60점 이상 |

#### 실습 과제

**1. PC 분해 조립 및 시스템 점검**
- 실제 PC를 분해하고 조립하는 실습
- 각 부품의 정상 작동 여부 확인
- 바이오스(BIOS) 설정

**2. PC 진단과 처방**
- 하드웨어 고장 진단
- 소프트웨어 문제 해결
- 최적화 작업 수행

**3. 하드웨어 업그레이드**
- 메모리, 저장장치 등의 부품 교체
- 성능 향상을 위한 업그레이드 방안 제시

**4. PC 네트워크 구축**
- 네트워크 케이블 연결
- IP 주소 설정
- 공유 폴더 및 프린터 설정

## 응시 정보

### 검정료

| 등급 | 과목 | 검정료 |
|------|------|--------|
| 2급 | 필기 | 43,000원 |
| 2급 | 실기 | 78,000원 |

### 결제 방법
- 무통장 입금
- 신용카드
- 온라인 이체

### 합격 기준

필기시험과 실기시험 **각각 100점 만점에 60점 이상** 취득 시 합격입니다.

:::info 필기와 실기 분리 응시 가능
필기시험과 실기시험은 별도로 응시할 수 있으며, 필기 합격 후 실기시험에 응시하거나 그 반대로도 가능합니다.
:::
