# 함수의 호출과 반환

## 🎪 함수 호출이란?

함수를 만드는 것만으로는 아무것도 실행되지 않습니다. 만든 함수를 **사용해야(호출해야)** 작동합니다.

**비유:** 요리 레시피(함수)를 쓰기만 하면 음식이 만들어지나요? 아니죠! **실제로 따라 만들어야(호출해야)** 음식이 완성됩니다.

## 📞 함수를 호출하는 방법

### 기본 형식

```c
함수이름(인수);  // 가장 간단한 호출
```

### 예제

```c
// 함수 정의
void greet() {
    printf("안녕하세요!\n");
}

// 메인 함수에서 호출
int main() {
    greet();   // 함수 호출 - 화면에 "안녕하세요!"가 출력됨
    return 0;
}
```

### 여러 번 호출하기

같은 함수를 여러 번 호출할 수 있습니다:

```c
void greet() {
    printf("안녕하세요!\n");
}

int main() {
    greet();   // 1번째 호출
    greet();   // 2번째 호출
    greet();   // 3번째 호출
    return 0;
}
```

**실행 결과:**
```
안녕하세요!
안녕하세요!
안녕하세요!
```

## 📬 인수(Arguments) - 함수에 값 전달하기

### 인수란?

함수를 호출할 때 **함수에게 전달하는 값**을 인수라고 합니다.

```c
add(5, 3);  // 5와 3은 인수입니다
//  ^^  ^
//  인수1  인수2
```

### 매개변수 vs 인수 (쉽게 이해하기)

```c
// 함수 정의할 때 사용하는 값 = 매개변수
int add(int a, int b) {  // a, b가 매개변수
    return a + b;
}

// 함수 호출할 때 사용하는 값 = 인수
int result = add(5, 3);  // 5, 3이 인수
```

**비유:**
- 📝 **매개변수** = 우체국 송장에 적힌 "받는 사람"
- 📦 **인수** = 실제로 배송되는 물건

### 인수의 규칙

```c
int add(int a, int b) {
    // 매개변수 2개를 정의함
    return a + b;
}

int main() {
    // ❌ 인수 1개만? 에러 발생!
    add(5);
    
    // ✅ 인수 2개 - 올바름
    add(5, 3);
    
    // ❌ 인수 3개? 에러 발생!
    add(5, 3, 2);
    
    return 0;
}
```

**규칙:** 매개변수 개수 = 인수 개수 (정확히 일치해야 함)

### 변수를 인수로 전달하기

```c
int add(int a, int b) {
    return a + b;
}

int main() {
    int num1 = 5;
    int num2 = 3;
    
    // 변수를 인수로 전달
    int result = add(num1, num2);  // num1(5), num2(3)를 전달
    
    printf("결과: %d\n", result);  // 결과: 8
    return 0;
}
```

## 🎁 반환값(Return Value) - 함수의 결과 받기

### 반환값이란?

함수가 계산을 마친 후 **돌려주는 값**을 반환값이라고 합니다.

```c
int add(int a, int b) {
    return a + b;  // a + b의 값을 돌려줍니다
}

int result = add(5, 3);  // 반환값 8을 받습니다
//  ^^^^^^
//  반환값을 저장할 변수
```

### 반환값을 변수에 저장하기

```c
int add(int a, int b) {
    return a + b;
}

int main() {
    // 반환값을 변수에 저장
    int result = add(5, 3);
    
    printf("결과: %d\n", result);  // 결과: 8
    return 0;
}
```

**실행 순서:**
```
1. add(5, 3) 호출
2. 함수 내에서 5 + 3 = 8 계산
3. 8을 반환 (return 8;)
4. 반환된 8을 result 변수에 저장
5. printf로 8 출력
```

### 반환값을 직접 사용하기

변수에 저장하지 않고 바로 사용할 수도 있습니다:

```c
int add(int a, int b) {
    return a + b;
}

int main() {
    // 방법 1: 변수에 저장 후 사용
    int result = add(5, 3);
    printf("결과: %d\n", result);
    
    // 방법 2: 직접 사용
    printf("결과: %d\n", add(5, 3));
    
    // 방법 3: 더하기에 사용
    int final = add(5, 3) + 10;  // (5+3) + 10 = 18
    
    return 0;
}
```

## 🔄 호출 과정 상세 분석

함수가 호출되었을 때 정확히 무엇이 일어나는지 단계별로 살펴보겠습니다.

### 코드

```c
int add(int a, int b) {
    int result = a + b;
    return result;
}

int main() {
    int num1 = 5;
    int num2 = 3;
    int sum = add(num1, num2);  // ← 여기서 호출
    printf("%d\n", sum);
    return 0;
}
```

### 실행 흐름 (단계별)

```
[1] main 함수 시작
    ↓
[2] num1 = 5 저장
    ↓
[3] num2 = 3 저장
    ↓
[4] add(num1, num2) 호출 ← 함수 호출!
    │
    └─→ [5] add 함수로 이동
        ↓
        [6] a = 5 (num1 값)
        [7] b = 3 (num2 값)
        ↓
        [8] result = 5 + 3 = 8
        ↓
        [9] return 8 ← 함수 종료, 결과 돌려줌
        │
        └─→ [10] 다시 main으로 돌아옴
            ↓
[11] sum = 8 저장
    ↓
[12] printf로 8 출력
    ↓
[13] main 함수 종료
```

## ⚠️ 흔한 실수들

### 실수 1: 괄호를 빠뜨리기

```c
void greet() {
    printf("안녕하세요!\n");
}

int main() {
    greet;      // ❌ 틀림! 괄호가 없음
    greet();    // ✅ 맞음
    return 0;
}
```

**문제:** 괄호가 없으면 함수가 호출되지 않습니다!

### 실수 2: 인수 개수 맞지 않기

```c
int add(int a, int b) {  // 매개변수 2개
    return a + b;
}

int main() {
    add(5);        // ❌ 인수 1개? 에러!
    add(5, 3);     // ✅ 인수 2개 - 맞음
    add(5, 3, 2);  // ❌ 인수 3개? 에러!
    return 0;
}
```

### 실수 3: 반환값이 없는데 받으려고 하기

```c
void print_hello() {
    printf("Hello!\n");
    // return이 없음 (void이므로)
}

int main() {
    int result = print_hello();  // ❌ 에러!
    // void 함수는 반환값이 없으므로 변수에 저장할 수 없음
    
    print_hello();  // ✅ 이렇게 사용
    return 0;
}
```

## 🎬 실제 예제

### 예제 1: 두 수의 합 구하기

```c
#include <stdio.h>

// 함수 선언 (원형)
int add(int a, int b);

int main() {
    // 변수 선언
    int num1 = 10;
    int num2 = 20;
    int sum;
    
    // 함수 호출 - num1과 num2를 인수로 전달
    sum = add(num1, num2);
    
    // 결과 출력
    printf("10 + 20 = %d\n", sum);
    
    return 0;
}

// 함수 정의
int add(int a, int b) {
    // 받은 값 두 개를 더함
    return a + b;
}
```

**실행 결과:**
```
10 + 20 = 30
```

**코드 분석:**
- 메인에서 `add(10, 20)` 호출
- 함수에서 10 + 20 = 30 계산
- 30을 반환
- sum에 30 저장
- 30 출력

### 예제 2: 학생 정보 출력 함수

```c
#include <stdio.h>

void print_student(char name[], int age) {
    printf("이름: %s\n", name);
    printf("나이: %d\n", age);
}

int main() {
    // 함수 호출 1
    print_student("김철수", 20);
    printf("\n");
    
    // 함수 호출 2
    print_student("이영희", 19);
    
    return 0;
}
```

**실행 결과:**
```
이름: 김철수
나이: 20

이름: 이영희
나이: 19
```

**코드 분석:**
- void 함수 (반환값 없음)
- 2개의 인수를 받음 (이름, 나이)
- 받은 정보를 출력하기만 함

## 📌 핵심 정리

:::note 꼭 기억하세요!
1. **호출** = 함수를 사용하기 위해 함수명() 형식으로 적기
2. **인수** = 호출할 때 전달하는 값들
3. **매개변수** = 함수 정의할 때 받을 값들
4. **반환값** = 함수가 돌려주는 결과
5. **규칙** = 매개변수 개수 = 인수 개수 (정확히 같아야 함)
:::
