# 함수의 구조와 선언

## 🏗️ 함수의 기본 구조

C언어의 함수는 다음과 같은 형태입니다:

```c
반환형 함수이름(매개변수) {
    // 함수의 몸통 (중괄호 내부)
    // 여기에 실행할 코드를 작성합니다
    return 돌려줄값;  // 결과를 돌려줍니다
}
```

각 부분을 자세히 살펴보겠습니다.

## 🔤 1️⃣ 반환형 (Return Type)

### 반환형이란?

함수가 작업을 마친 후 **돌려줄 값의 자료형(타입)**을 의미합니다.

```c
int add(int a, int b) {
    //  ^^^
    // 이것이 반환형입니다 = 정수(int)를 반환하겠다는 뜻
    return a + b;  // 정수를 반환합니다
}
```

### 반환형의 종류

```c
int function_name()     // 정수를 반환하는 함수
float function_name()   // 실수를 반환하는 함수
double function_name()  // 더 정밀한 실수를 반환하는 함수
char function_name()    // 문자 하나를 반환하는 함수
void function_name()    // 반환값이 없는 함수
```

### ⚠️ void 반환형 - 특별한 경우

만약 반환할 값이 **없다**면 `void`를 사용합니다:

```c
void print_message() {
    // 반환형이 void = 값을 돌려주지 않음
    printf("안녕하세요!\n");
    // return이 없어도 됩니다
}
```

**비유:** 자판기에서 선택만 하고 아무것도 받지 않는 것처럼, 함수가 작업만 하고 값을 돌려주지 않습니다.

## 📝 2️⃣ 함수 이름 (Function Name)

### 함수 이름 정하기

함수의 이름은 매우 중요합니다. **함수가 무엇을 하는지 명확하게 알 수 있는 이름**이 좋습니다:

```c
// ✅ 좋은 함수 이름 (함수의 기능이 명확함)
int get_sum(int a, int b)         // 합을 구한다
void print_student_info()         // 학생 정보를 출력한다
int calculate_age(int birth_year) // 나이를 계산한다
float get_average()               // 평균을 구한다

// ❌좋지 않은 함수 이름 (무엇을 하는지 불명확함)
void function()        // 뭘 하는 함수인지 몰라요
int test()             // 테스트? 뭘 테스트하나요?
void do_something()    // '뭔가'한다는 게 뭐예요?
```

### 함수 이름 규칙

```c
// ✅ 올바른 규칙
- 영문자로 시작 (한글은 안 됨)
- 숫자나 언더바(_)는 사용 가능하지만 처음에는 안 됨
- 대소문자를 구별합니다 (Add ≠ add)

// ✅ 사용 가능한 이름들
add()
getAge()
print_message()
calculateSum123()
```

## 🎯 3️⃣ 매개변수 (Parameter)

### 매개변수란?

**함수가 외부로부터 받을 값들**을 적는 부분입니다.

```c
int add(int a, int b) {
    //   ^^^^^^^^^^^^^^
    //   이 부분이 매개변수입니다
    //   정수 a, 정수 b를 받겠다는 뜻
    return a + b;
}
```

### 매개변수의 역할

```
호출하는 곳에서 값을 보냄 → 함수가 받음(매개변수) → 함수 내에서 사용
```

### 매개변수가 없는 경우

매개변수가 없으면 **괄호는 비어있습니다**:

```c
void print_message() {
    //               ↑
    //         괄호가 비어있음 = 받을 값이 없음
    printf("안녕하세요!\n");
}
```

### 매개변수가 여러 개인 경우

```c
// 매개변수가 2개
int add(int a, int b) {
    return a + b;
}

// 매개변수가 3개
int sum_three(int x, int y, int z) {
    return x + y + z;
}

// 매개변수가 0개
void say_hello() {
    printf("Hello!\n");
}
```

## 🎁 4️⃣ 함수의 몸통 (Function Body)

### 중괄호 { } 사이의 코드

함수가 실제로 **무엇을 할 것인지** 작성하는 부분입니다:

```c
int add(int a, int b) {
    // 시작 중괄호
    
    int result;           // 변수 선언
    result = a + b;       // 계산 수행
    
    // 끝 중괄호
    return result;        // 결과 반환
}
```

## 🔄 5️⃣ return 문 (반환문)

### return의 역할

함수의 작업이 끝났을 때 **결과값을 돌려주고 함수를 종료**합니다:

```c
int get_number() {
    int num = 10;
    return num;  // num 값을 돌려주고 함수 종료
    
    // return 이후의 코드는 실행되지 않음!
    printf("This won't print");  // 실행 안 됨
}
```

### return 사용 규칙

```c
// ✅ 반환형과 같은 자료형을 반환해야 함
int function() {
    return 5;        // int를 반환하므로 ✅ 맞음
    return 5.5;      // float를 반환하지만 int로 선언되면 ❌ 문제있음
}

// ✅ void 함수는 return값이 없음
void print_text() {
    printf("Hello\n");
    return;  // 또는 return을 쓰지 않아도 됨
}
```

## 📚 전체 함수 구조 정리

```c
int add(int a, int b) {
//  │   │  │  │    │
//  │   │  │  │    └─ 매개변수 2: 정수 b
//  │   │  │  └─────── 매개변수 1: 정수 a
//  │   │  └────────── 매개변수 시작/끝
//  │   └───────────── 함수 이름
//  └────────────────── 반환형: 정수
    
    int result = a + b;
    return result;      // 정수를 반환
}
```

## 🎬 실제 예제로 이해해보기

### 예제 1: 간단한 덧셈 함수

```c
int add(int num1, int num2) {
    return num1 + num2;
}
```

**분석:**
- **반환형:** `int` → 정수를 반환합니다
- **함수명:** `add` → 더하기 함수입니다
- **매개변수:** `int num1, int num2` → 두 개의 정수를 받습니다
- **동작:** num1과 num2를 더한 결과를 반환합니다

### 예제 2: 메시지를 출력하는 함수

```c
void greet(char name[]) {
    printf("안녕하세요, %s님!\n", name);
}
```

**분석:**
- **반환형:** `void` → 값을 반환하지 않습니다
- **함수명:** `greet` → 인사하는 함수입니다
- **매개변수:** `char name[]` → 문자열 하나를 받습니다
- **동작:** 받은 이름으로 인사 메시지를 출력합니다

### 예제 3: 나이를 계산하는 함수

```c
int calculate_age(int birth_year) {
    return 2024 - birth_year;
}
```

**분석:**
- **반환형:** `int` → 나이(정수)를 반환합니다
- **함수명:** `calculate_age` → 나이를 계산합니다
- **매개변수:** `int birth_year` → 태어난 연도를 받습니다
- **동작:** 2024년에서 출생년도를 뺀 값(나이)을 반환합니다

## 📌 핵심 정리

:::note 꼭 기억하세요!
1. **반환형** = 돌려줄 값의 자료형
2. **함수명** = 함수가 무엇을 하는지 알 수 있는 이름
3. **매개변수** = 함수가 받을 값들
4. **중괄호 { }** = 함수가 할 일이 적혀있는 부분
5. **return** = 결과를 돌려주고 함수 종료
:::
