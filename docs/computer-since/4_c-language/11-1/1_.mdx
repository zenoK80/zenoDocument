# 함수의 기본 개념

## 🎯 수학에서의 함수와 프로그래밍의 함수

프로그래밍을 처음 배우시는 분들을 위해 함수 개념을 쉽게 설명하겠습니다.

### 📚 수학에서의 함수

여러분들이 이미 수학 시간에 배웠을 함수를 먼저 떠올려보세요.

수학의 함수는 다음과 같은 형태입니다:
```
y = f(x) = 3x + 5
```

만약 x에 **2를 대입**하면:
```
y = 3 × 2 + 5 = 11
```

**입력값(x)을 넣으면** → **어떤 계산을 한 후** → **결과값(y)을 얻는다**는 것이 함수입니다.

### 💻 프로그래밍에서의 함수

프로그래밍의 함수도 정확히 같은 개념입니다!

**프로그래밍의 함수는:**
- 값을 받아서 (입력)
- 어떤 작업을 처리하고 (처리)
- 결과를 돌려준다 (출력)

이렇게 3가지 역할을 하는 **작은 프로그램**이라고 생각하면 됩니다.

## 🏠 함수는 왜 필요할까?

프로그램이 커질수록 코드의 양이 엄청나게 늘어납니다.

```
프로그램의 규모가 커지면:
큰 프로그램 = 작은 함수들이 모여서 만들어짐
```

**예를 들어보겠습니다:**
- 1,000줄 짜리 프로그램보다
- 100줄짜리 함수 10개를 조합하는 것이
- 훨씬 더 효율적이고 읽기 쉽습니다!

## 📖 우리가 이미 사용한 함수들

놀랍게도 여러분들은 이미 함수를 사용하고 있었습니다!

### printf 함수
```c
printf("안녕하세요");  // 화면에 글씨를 출력해주는 함수
```
- **역할:** 문자열이나 변수 값을 화면에 출력

### scanf 함수
```c
scanf("%d", &num);  // 사용자로부터 입력을 받는 함수
```
- **역할:** 사용자로부터 값을 입력받음

### main 함수
```c
int main() {
    // 프로그램의 모든 내용이 여기에 들어갑니다
    return 0;
}
```
- **역할:** 프로그램이 시작되는 가장 중요한 함수

## 🎪 함수의 특징: 가로() 괄호

C언어에서 어떤 단어 뒤에 **가로() 괄호**가 붙어있으면 그것은 **함수**라고 생각하면 됩니다!

```c
main()      // ← 함수예요
printf()    // ← 함수예요
scanf()     // ← 함수예요

변수명      // ← 함수가 아니에요 (괄호가 없음)
```

## 🧠 함수 이해하기: 자판기 비유

함수를 **자판기**라고 생각해보세요:

```
1. 동전을 넣는다 (입력)
   ↓
2. 자판기가 작동한다 (처리)
   ↓
3. 음료수가 나온다 (출력)
```

**프로그래밍의 함수도 똑같습니다:**

```
1. 값을 넣는다 (입력)
   ↓
2. 함수가 작동한다 (처리)
   ↓
3. 결과가 나온다 (출력)
```

## 🎁 함수 사용의 가장 큰 이유

### 1️⃣ 반복되는 코드를 줄일 수 있습니다

**함수 없이:** 같은 코드를 100번 써야 함
```c
printf("배움 C언어\n");
printf("배움 C언어\n");
printf("배움 C언어\n");
// ... 100번 반복 😫
```

**함수를 사용하면:** 함수를 100번 호출하면 됨
```c
print_message();  // 한 줄로 끝!
print_message();
print_message();
```

### 2️⃣ 코드를 읽기 쉽게 만듭니다

메인 함수에 모든 기능을 넣으면:
- 어디에 어떤 기능이 있는지 찾기 어려움
- 코드가 너무 복잡해 보임

함수로 나누면:
- 각 함수의 이름만 봐도 무엇을 하는지 알 수 있음
- 전체 프로그램의 흐름을 한눈에 파악 가능

### 3️⃣ 유지 보수가 쉬워집니다

만약 기능을 수정해야 한다면:

**함수 없이:** 모든 코드(100곳)를 다 수정
```c
// 1번 수정
// 2번 수정
// ... 100번 수정 😫
```

**함수 사용:** 함수 한 곳만 수정
```c
// 함수 내부만 수정하면 됨!
// 호출하는 곳들은 자동으로 반영됨 😊
```

## 📌 핵심 포인트

:::note 기억하세요!
1. **함수 = 입력 → 처리 → 출력**
2. **괄호()가 있으면 함수입니다**
3. **반복되는 코드는 함수로 만들어요**
4. **함수로 코드를 나누면 더 효율적입니다**
:::
