# 자료구조(Data Structure)란?

## 개요

자료구조는 **컴퓨터 과학의 핵심 과목**으로, 데이터를 효율적으로 저장하고 관리하며 활용하기 위한 방법을 연구하는 학문입니다.  
프로그램이 데이터를 어떻게 조직하고 접근할지를 결정하는 중요한 개념들을 다룹니다.

:::tip 핵심 포인트
좋은 자료구조 선택 = 빠른 프로그램 + 적은 메모리 사용
:::

---

## 왜 배워야 하나요?

자료구조를 배우는 이유는 다음과 같습니다:

- **효율적인 프로그램 작성**: 적절한 자료구조를 선택하면 프로그램의 실행 속도와 메모리 사용량을 크게 개선할 수 있습니다
- **문제 해결 능력 향상**: 복잡한 문제를 자료구조를 활용해 체계적으로 접근할 수 있게 됩니다
- **알고리즘 이해의 기초**: 대부분의 알고리즘은 특정 자료구조를 기반으로 동작합니다

---

## 주요 학습 주제

자료구조 과목에서는 다음과 같은 내용을 배웁니다:

### 기본 개념
- 시간 복잡도와 공간 복잡도
- Big-O 표기법
- 추상 자료형(ADT)

### 선형 자료구조
- 배열 (Array)
- 연결 리스트 (Linked List)
- 스택 (Stack)
- 큐 (Queue)

### 비선형 자료구조
- 트리 (Tree)
- 그래프 (Graph)
- 힙 (Heap)

### 고급 자료구조
- 해시 테이블 (Hash Table)
- 우선순위 큐
- 트라이 (Trie)
