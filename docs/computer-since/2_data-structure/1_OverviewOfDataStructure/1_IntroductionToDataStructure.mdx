# 자료구조의 개요

## 자료구조란 무엇인가?

자료구조(Data Structure)는 **데이터를 효율적으로 저장하고 관리하기 위한 구조**입니다. 

자료구조가 처음 등장한 배경은 **일상생활의 많은 문제를 해결하기 위해서**였습니다. 우리가 책을 정리하거나, 은행에서 대기하거나, 네비게이션을 사용하는 것들이 모두 자료구조와 밀접하게 연관되어 있습니다.

:::tip 핵심 개념
자료구조를 배우는 목적은 **일상생활의 문제를 효율적으로 해결**하는 것입니다.

## 일상생활 속 자료구조

우리 주변에서 쉽게 찾아볼 수 있는 자료구조의 예시들입니다:

### 실생활 vs 자료구조 대응

| 일상생활 | 자료구조 | 설명 |
|---------|---------|------|
| 접시 쌓기 | 스택 (Stack) | 마지막에 올린 접시를 먼저 꺼냄 |
| 마트 계산대 줄 | 큐 (Queue) | 먼저 온 사람이 먼저 계산 |
| 영화관 대기 | 큐 (Queue) | 선착순으로 입장 |
| 버킷리스트 | 리스트 (List) | 항목들을 순서대로 나열 |
| 영어 사전 | 해시 테이블 | 단어를 빠르게 검색 |
| 지하철 노선도 | 그래프 (Graph) | 역들의 연결 관계 표현 |
| 컴퓨터 폴더 구조 | 트리 (Tree) | 계층적 구조 표현 |

### 주요 자료구조 용어

**스택 (Stack)**
- 접시를 쌓는 것처럼 데이터를 쌓아 올리는 구조
- 마지막에 넣은 것을 먼저 꺼냄

**큐 (Queue)**
- 줄을 서는 것처럼 순서대로 처리하는 구조
- 먼저 넣은 것을 먼저 꺼냄

**리스트 (List)**
- 데이터를 일렬로 나열하는 구조
- 순서대로 항목을 관리

**그래프 (Graph)**
- 지도처럼 여러 지점들의 연결 관계를 표현
- 네트워크, 경로 찾기 등에 활용

**트리 (Tree)**
- 폴더 구조처럼 계층적으로 데이터를 관리
- 상위-하위 관계를 명확하게 표현

:::info 용어가 낯설어도 괜찮아요!
지금은 이런 용어들이 생소할 수 있습니다.  
중요한 건 **일상생활의 문제를 자료구조로 해결할 수 있다**는 관점을 이해하는 것입니다.
:::
