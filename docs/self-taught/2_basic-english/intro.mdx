# 독학사 1단계 영어란?

## 개요

독학사 1단계 영어는 **독학학위제 교양과정의 핵심 과목**으로, 기본적인 영어 실력을 평가하고 향상시키는 것을 목표로 합니다.

일상생활과 학업에 필요한 영어 능력을 종합적으로 다루며, 어휘, 문법, 독해, 영작, 생활영어까지 폭넓게 학습합니다.

:::tip 핵심 포인트
어휘 + 문법 + 독해 = 실용적인 영어 능력 향상
:::

---

## 왜 배워야 하나요?

독학사 1단계 영어를 배우는 이유는 다음과 같습니다:

- **기초 영어 능력 강화**: 대학 교양 수준의 영어 실력을 체계적으로 다집니다
- **실생활 활용**: 일상 대화부터 학술적 글까지 다양한 영어 텍스트를 이해할 수 있습니다
- **학위 취득**: 독학학위제 학사 학위 취득을 위한 필수 과정입니다
- **종합적 평가**: 단순 암기가 아닌 실제 활용 능력을 측정합니다

---

## 주요 학습 영역

독학사 1단계 영어는 5개의 대영역으로 구성되어 있습니다:

### 1. 어휘 및 숙어

- 단어의 의미 파악
- 유의어, 반의어 이해
- 관용적 표현 학습
- 문맥에 맞는 어휘 선택

### 2. 문법과 구조

**가. 구두점**
- 문장 부호의 올바른 사용
- 구두점을 통한 의미 파악

**나. 문법**
- 시제와 동사 활용
- 품사와 문장 구조
- 전치사 활용
- 비교급과 최상급
- 가정법과 조동사

### 3. 독해력

**가. 대의 파악**
- 글의 전체적인 내용 이해
- 중심 생각 파악

**나. 주제 파악**
- 글의 핵심 주제 찾기
- 제목 추론하기

**다. 세부 사항 파악**
- 구체적인 정보 찾기
- 사실과 의견 구분

**라. 결론 도출**
- 글의 논리적 결론 추론
- 암시된 의미 파악

**마. 논조 이해**
- 필자의 태도와 어조 파악
- 글의 목적 이해

**바. 글의 상호관계 분석**
- 문장 간의 논리적 연결
- 단락 간의 관계 파악

**사. 영문 국역**
- 영어를 자연스러운 한국어로 번역
- 문맥을 고려한 정확한 의미 전달

### 4. 영작

- 한국어를 영어로 올바르게 표현
- 문법적으로 정확한 문장 작성
- 자연스러운 영어 표현 활용

### 5. 생활영어

- 일상 대화 표현
- 상황별 적절한 응답
- 실용적인 의사소통 능력

## 시험 문제 유형

독학사 1단계 영어 시험은 다음과 같은 유형으로 출제됩니다:

### 어휘 문제
- 밑줄 친 단어의 유의어 고르기
- 문맥에 맞는 단어 선택하기

### 문법 문제
- 빈칸에 적절한 전치사, 접속사 넣기
- 어법상 틀린 문장 찾기
- 문법적으로 올바른 문장 고르기

### 독해 문제
- 지문을 읽고 내용 이해하기
- 단어의 의미 추론하기
- 글의 주제와 요지 파악하기

### 회화 문제
- 대화의 빈칸에 적절한 응답 고르기
- 상황에 맞는 표현 선택하기

### 영작 문제
- 한국어를 영어로 올바르게 옮기기
- 속담의 의미 파악하기

## 학습 전략

효과적인 독학사 영어 학습을 위한 팁입니다:

1. **어휘력 강화**: 매일 꾸준히 단어를 암기하고 예문과 함께 학습하세요
2. **문법 정리**: 기본 문법 개념을 체계적으로 정리하고 문제를 통해 반복 연습하세요
3. **독해 연습**: 다양한 지문을 읽으며 독해 속도와 이해력을 향상시키세요
4. **실전 연습**: 기출문제와 모의고사를 풀어보며 시험 유형에 익숙해지세요
5. **생활영어 활용**: 실제 상황을 상상하며 회화 표현을 익히세요
