# 통계의 사용: 데이터를 다루는 두 가지 관점

통계학은 데이터를 다루는 목적에 따라 크게 **기술통계학**과 **추측통계학**으로 나뉩니다. 시험에서는 이 둘의 정의와 예시를 구분하는 문제가 자주 출제됩니다.

## 1. 기술통계학 Descriptive Statistics

수집된 데이터를 **요약하고 정리**하여, 데이터의 **현재 상태**를 알기 쉽게 설명하는 분야입니다.

### 특징과 목적

- **목적**: 복잡한 데이터를 한눈에 파악하기 위함 - 묘사
- **범위**: 이미 수집된 데이터에만 적용
- **오류**: 표본오차와 무관 - 표본 전체를 분석

### 주요 활동

**시각화**
- 막대그래프, 원그래프, 히스토그램 작성
- 데이터의 분포와 패턴을 시각적으로 표현

**요약값 계산**
- **평균**: 데이터의 중심 위치
- **분산, 표준편차**: 데이터의 퍼짐 정도
- **중앙값, 최빈값**: 다른 대표값

### 사용 예시

- 2023년 대한민국 출생아 수는 약 23만 명이다.
- A반 학생들의 영어 성적 평균은 75점이다.
- 서울시 평균 기온은 15도이다.

:::caution 주의
여기서는 미래를 예측하거나, 보이지 않는 것을 추측하지 않습니다. 있는 그대로만 보여줍니다.
:::

## 2. 추측통계학 Inferential Statistics

표본 일부을 분석하여 모집단 전체의 특성을 **추측하고 예측**하는 분야입니다.

### 특징과 목적

- **목적**: 불확실한 사실에 대해 확률을 근거로 의사결정을 내리기 위함
- **범위**: 표본으로부터 모집단 전체를 추론
- **오류**: 표본오차 발생 가능 - 신뢰도로 관리

### 주요 활동

**추정 Estimation**
- 전체 국민의 지지율은 40% 내외일 것이다.
- 경주시 주민의 평균 월소득은 300만 원 정도일 것이다.

**가설검정 Hypothesis Testing**
- 이 신약은 효과가 있다고 볼 수 있다.
- A 상품과 B 상품의 품질에 유의한 차이가 있는가?

**예측 Prediction**
- 내년 경제 성장률은 2%로 예상된다.
- 다음 월의 판매량은 증가할 것으로 예측된다.

### 사용 예시

- **선거 출구조사**: 1,000명을 조사해 전체 투표 결과 추측
- **신약 임상실험**: 일부 환자를 대상으로 실험해 약의 효과 판단
- **품질 관리**: 전체 제품 중 일부 표본만 검사해 불량률 추정

## 3. 기술통계학과 추측통계학의 비교

| 구분 | 기술통계학 | 추측통계학 |
| :--- | :--- | :--- |
| **정의** | 데이터를 정리·요약 | 표본으로 모집단 추론 |
| **대상** | 수집된 데이터 전체 | 표본 데이터 |
| **목적** | 현재 상태 파악 | 미래 예측 및 의사결정 |
| **도구** | 표, 그래프, 평균, 분산 | 추정, 가설검정, 예측 |
| **불확실성** | 없음 | 있음 - 확률로 표현 |
| **신뢰도** | 100% 정확 | 95% 신뢰수준 등으로 표현 |

## 4. 통계 분석의 흐름

통계 분석은 논리적인 순서에 따라 진행됩니다.

**단계 1: 문제 제기**
- 무엇을 알고 싶은가? - 연구 목적 설정
- 조사할 모집단 정의

**단계 2: 자료 수집**
- 모집단을 정의하고 표본을 추출
- 설문조사, 관찰, 실험 등으로 데이터 수집

**단계 3: 자료 정리 기술통계**
- 수집된 데이터를 표나 그래프로 요약
- 기본 통계량 평균, 분산 등 계산
- 데이터의 특성 파악

**단계 4: 자료 분석 추측통계**
- 표본 정보를 통해 모집단의 특성을 추론
- 가설검정 실시
- 신뢰도 설정

**단계 5: 결과 해석**
- 분석 결과를 바탕으로 결론 도출
- 의사결정 수행
- 건의사항 제시

## 5. 통계 분석의 중요성

기술통계학과 추측통계학은 서로 보완적인 관계입니다. 정확한 결론을 도출하기 위해서는 두 분야를 모두 이해하고 적절히 활용해야 합니다. 단순히 데이터를 요약하는 것뿐만 아니라, 그 데이터로부터 신뢰할 수 있는 결론을 이끌어내는 것이 통계학의 진정한 역할입니다.
