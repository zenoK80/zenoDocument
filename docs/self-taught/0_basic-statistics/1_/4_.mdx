# 측정척도: 숫자와 데이터의 수준을 구분하는 방법

## 🎯 측정척도란 무엇인가?

**쉽게 말하면**: 우리가 조사할 때 사용하는 **숫자나 데이터의 품질 수준**을 나누는 방식입니다.

예를 들어보겠습니다:
- 어떤 숫자는 **단순히 구분**하기 위한 것 - 예: 옷의 사이즈 S, M, L
- 어떤 숫자는 **순서**가 있는 것 - 예: 학점 A, B, C
- 어떤 숫자는 **정확한 차이**를 나타내는 것 - 예: 온도 10도, 20도
- 어떤 숫자는 **비율**까지 가능한 것 - 예: 키 150cm, 300cm는 정확히 2배

이런 **4가지 수준**을 이해하는 게 측정척도입니다!

:::info 왜 중요한가?
통계 분석을 할 때 **어떤 계산이 가능한지**가 결정되기 때문입니다. 
예를 들어 학점 A, B, C의 평균을 낼 수는 있지만 **통계적으로 의미가 없습니다**. 
하지만 키의 평균은 의미 있습니다!
:::

## 📊 Level 1: 명목척도 (Nominal Scale) - "이름 붙이기"

### 🔍 명목척도의 정의

**명목척도란**: 숫자나 문자를 **단순히 분류, 구분, 이름붙이기** 위해 사용하는 가장 기본적인 척도입니다.

**쉽게 말하면**: "이것은 A이고, 이것은 B이다"라고 **구분만 하는 것**입니다.

### 📌 명목척도의 특징

**1. 순서가 없다**
- 예: 성별에서 남성 1번, 여성 2번이라고 했을 때, 2번이 1번보다 크거나 중요한 게 아님
- 그냥 구분 기호일 뿐

**2. 숫자도 단순한 기호다**
- 예: 축구 등번호 10번이 5번보다 2배 잘 한다는 뜻이 아님
- 그냥 선수를 구분하는 기호

**3. 연산이 불가능하다**
- 남성 + 여성 = ? → 말도 안 됨
- 혈액형 A + 혈액형 B = ? → 불가능
- **더하기, 빼기, 곱하기, 나누기 모두 불가능**

**4. 할 수 있는 것은 '같은가, 다른가'뿐이다**
- "이 사람은 남성인가? 예/아니오"
- "이 사람은 서울에 사는가? 예/아니오"

### 💡 명목척도의 실생활 예시

**성별**
```
남 = 1번
여 = 2번

분석 예: "우리 학교에 남학생 300명, 여학생 200명이 있다"
불가능한 분석: "평균 성별은 1.4다" (말도 안 됨!)
```

**종교**
```
불교 = A
기독교 = B
천주교 = C
무교 = D

분석 예: "조사 대상 중 불교 신자가 40명이다"
불가능한 분석: "평균 종교는 B + 0.3이다" (말도 안 됨!)
```

**혈액형**
```
A형 = 1
B형 = 2
AB형 = 3
O형 = 4

분석 예: "직원 중 O형이 25%이다"
불가능한 분석: "우리 회사의 평균 혈액형은 2.5다" (말도 안 됨!)
```

**지역**
```
서울 = 1
경기 = 2
인천 = 3
대전 = 4

분석 예: "서울에 사는 사람이 500명이다"
불가능한 분석: "평균 지역은 2.1이다" (의미 없음!)
```

### 📈 명목척도로 할 수 있는 분석

**1. 빈도 계산 Frequency**
- "우리 반 학생 중 남자는 15명, 여자는 20명이다"
- 단순히 몇 명씩 있는지만 센다

**2. 비율 계산 Percentage**
- "남자는 전체의 43%, 여자는 57%이다"
- 전체에서 차지하는 비율

**3. 최빈값 Mode**
- "가장 많은 혈액형은 O형이다"
- 가장 빈번하게 나타나는 항목

**4. 교차 분석 Cross Tabulation**
```
          남   여
A형       50   40
B형       30   45
O형       60   55
```
- 두 개의 분류 항목을 섞어서 분석

### ❌ 명목척도로 절대 할 수 없는 것

- **평균**: 성별의 평균 = 불가능
- **표준편차**: 혈액형의 표준편차 = 불가능
- **더하기**: A형 + B형 = 불가능
- **순서 비교**: "서울이 부산보다 크다" = 틀린 말

### 🎓 시험에 나올 핵심

**Q. 다음 중 명목척도의 특징이 아닌 것은?**
1) 순서가 없다
2) 더하기가 불가능하다
3) 평균을 낼 수 있다 → **정답!**
4) 숫자는 단순 기호이다

## 📊 Level 2: 서열척도 (Ordinal Scale) - "순위 매기기"

### 🔍 서열척도의 정의

**서열척도란**: 명목척도에 **순서나 순위**를 추가한 척도입니다.

**쉽게 말하면**: "A가 B보다 크고, B가 C보다 크고..."라는 **순서 관계**가 있는 데이터입니다.

### 📌 서열척도의 특징

**1. 순서가 있다**
- 예: 학점 A > B > C > D > F
- 명확한 순서가 있음

**2. 하지만 간격이 일정하지 않다** ⭐⭐⭐ (매우 중요!)
- A와 B 사이의 "거리"와 B와 C 사이의 "거리"가 다를 수 있음
- 예를 들어:
  - 1등 학생과 2등 학생의 점수 차이: 95점 - 92점 = 3점
  - 2등 학생과 3등 학생의 점수 차이: 92점 - 85점 = 7점
  - **같은 "순서"이지만 점수 차이는 다르다!**

**3. 더하기, 빼기가 의미 없다**
- 학점 A + 학점 B = ? → 의미 없음
- 비록 숫자로 표현할 수 있어도(A=5, B=4) 더하는 것은 의미 없음

**4. 순서 비교만 가능하다**
- "A가 B보다 낫다" - 가능!
- "A가 B의 몇 배다" - 불가능!

### 💡 서열척도의 실생활 예시

**학점**
```
A = 5점 (우수)
B = 4점 (좋음)
C = 3점 (보통)
D = 2점 (부족)
F = 1점 (미흡)

왜 서열척도일까?
- A > B > C > D > F라는 순서가 있다
- 하지만 A와 B의 차이가 B와 C의 차이와 같지 않다
- 대학에서 GPA 계산할 때 평균을 내는 이유?
  → 그냥 관례일 뿐, 통계적으로 완벽하진 않다!
```

**만족도 조사**
```
매우만족 = 5 (가장 높음)
만족     = 4
보통     = 3
불만족   = 2
매우불만족= 1 (가장 낮음)

예시: "이 제품에 대해 어느 정도 만족하나요?"
- 순서: 매우만족 > 만족 > 보통 > 불만족 > 매우불만족
- 하지만 "매우만족과 만족의 차이"와 "만족과 보통의 차이"가
  같지 않을 수 있다
```

**영화 평점**
```
⭐⭐⭐⭐⭐ = 5점 (최고!)
⭐⭐⭐⭐   = 4점 (좋음)
⭐⭐⭐    = 3점 (보통)
⭐⭐     = 2점 (별로)
⭐      = 1점 (형편없음)

5점과 4점 차이가
4점과 3점 차이와 같지 않을 수 있다!
```

**직급**
```
회장 > 사장 > 부사장 > 상무 > 이사 > 부장 > 과장 > 대리 > 사원

순서는 명확하지만:
- "사장 - 부사장"의 권한 차이
- "과장 - 대리"의 권한 차이
- 이 둘이 같지 않다!
```

**품질 등급**
```
최고급(A) > 1급(B) > 2급(C) > 3급(D)

순서는 있지만:
- 최고급과 1급의 차이가 얼마나 큰지 알 수 없음
- 숫자(4, 3, 2, 1)를 더하는 것은 의미 없음
```

### 📈 서열척도로 할 수 있는 분석

**1. 명목척도의 모든 것** (빈도, 비율, 최빈값)
- "우리 반 학생 중 A학점이 10명이다"
- "A학점 학생이 전체의 20%이다"

**2. 추가로: 중앙값 Median**
- "학생들의 중간 등급은 B이다"
- 즉, 50% 지점이 어디인가 = 중앙값

**3. 순위 기반 비교**
- "상위 30%의 학생들을 뽑았다"
- "하위 10%는 재시험을 본다"

### ❌ 서열척도로 절대 할 수 없는 것

- **평균**: A(5) + B(4) + C(3) / 3 = 4 → 수학적으로는 가능하지만 **통계적 의미 없음**
- **표준편차**: 마찬가지로 의미 없음
- **더하기**: A + B = ? 의미 없음
- **비율**: "A는 B의 1.25배 좋다" → 틀린 말

### ⚠️ 서열척도 vs 명목척도의 핵심 차이

```
명목척도      서열척도
========      ========
성별          학점
혈액형        만족도
종교          직급
지역          등급

명목: 순서 없음   서열: 순서 있음!
```

### 🎓 시험에 나올 핵심

**Q. 서열척도의 가장 큰 특징은?**
- A) 순서가 있다
- B) 평균을 낼 수 있다
- C) 모든 연산이 가능하다
- D) 같은 간격을 가진다

**정답: A) 순서가 있다**

**Q. 다음 중 서열척도는?**
1) 혈액형 - 아니오 (명목척도)
2) 키 - 아니오 (비율척도)
3) 학점 - **맞음! (서열척도)**
4) 종교 - 아니오 (명목척도)

## 🌡️ Level 3: 등간척도 (Interval Scale) - "일정한 간격의 숫자"

### 🔍 등간척도의 정의

**등간척도란**: 순서가 있으면서 **간격이 일정한** 척도입니다.

**쉽게 말하면**: "이 숫자들 사이의 거리가 항상 같다"는 의미입니다.

### 📌 등간척도의 특징

**1. 순서가 있다**
- 30도 > 20도 > 10도
- 명확한 크기 순서가 있음

**2. 간격이 일정하다** ⭐⭐⭐
- 10도에서 20도로 올라가는 것 = 20도에서 30도로 올라가는 것
- **둘 다 10도씩 올라감**
- 간격이 항상 동일

**3. 더하기, 빼기가 가능하다**
- 30도 - 20도 = 10도 (의미 있음!)
- 차이 계산이 가능

**4. 하지만 비율은 안 된다** ⭐⭐⭐ (매우 중요!)
- "20도는 10도의 2배 따뜻하다" → **틀린 말!!!**
- 왜? 절대영점이 없기 때문

### 🎯 절대영점이란?

**절대영점**: "완전히 없는 상태"를 나타내는 0입니다.

**예시:**
- **온도의 0도** = 열이 0도라고 해서 열이 없다는 뜻이 아님
  - 섭씨 0도 = 물이 어는 온도일 뿐
  - 절대적으로 "열이 없음"을 의미하지 않음
  - 그래서 "20도는 10도의 2배"라고 할 수 없음

- **시간의 0시** = 시간이 없다는 뜻이 아님
  - 자정을 0시라고 부를 뿐
  - 절대영점이 아님

### 💡 등간척도의 실생활 예시

**온도(섭씨)**
```
겨울: 10도
봄: 20도
여름: 30도

가능한 분석:
- "30도는 10도보다 20도 더 높다" ✓
- "여름은 봄보다 10도 더 따뜻하다" ✓
- 평균 기온: (10 + 20 + 30) / 3 = 20도 ✓

불가능한 분석:
- "30도는 10도의 3배 따뜻하다" ✗
  왜? 0도가 "온도가 없음"을 의미하지 않기 때문

왜 비율이 의미 없나?
- 아래 예를 보세요:
- 섭씨 20도 = 화씨로 68도
- 섭씨 10도 = 화씨로 50도
- 섭씨에서는 "20도는 10도의 2배"
- 하지만 화씨에서는 "68도는 50도의 1.36배"
- 같은 온도를 다르게 표현하면 배수가 바뀐다!
- 따라서 비율은 의미 없는 것!
```

**IQ 지수**
```
일반적 IQ: 90 ~ 110
영재 IQ: 130 이상

가능한 분석:
- "IQ 120은 IQ 100보다 20포인트 높다" ✓
- 평균 IQ: 계산 가능 ✓

불가능한 분석:
- "IQ 100은 IQ 50의 2배 똑똑하다" ✗
- IQ 0은 지능이 없다는 뜻이 아니기 때문
```

**시험 점수(일부 경우)**
```
학생 A: 90점
학생 B: 60점
학생 C: 30점

가능한 분석:
- "A는 B보다 30점 더 높다" ✓
- 평균: (90 + 60 + 30) / 3 = 60점 ✓

불완벽한 분석:
- "A는 C의 3배 잘했다" 
  → 이론상 가능하지만 교육적 의미는 약함
  → 0점이 "전혀 모름"을 완벽히 의미하지 않을 수 있기 때문
```

**연도**
```
2000년, 2010년, 2020년

가능한 분석:
- "2020년은 2010년보다 10년 뒤다" ✓
- 연도 차이 계산: 2020 - 2000 = 20년 ✓

불가능한 분석:
- "2020년은 2010년의 2배다" ✗
- "0년도"는 시간의 절대 시작점이 아니기 때문
```

### 📈 등간척도로 할 수 있는 분석

**1. 서열척도의 모든 것** (빈도, 비율, 중앙값, 순위)

**2. 추가로: 평균**
- "평균 온도는 20도이다" ✓
- 의미 있는 분석

**3. 표준편차**
- "온도의 표준편차는 5도이다" ✓
- 데이터가 얼마나 흩어져 있는지 알 수 있음

**4. 상관계수**
- "온도가 올라가면 판매량도 올라간다" ✓
- 두 변수의 관계 분석 가능

### ❌ 등간척도로 절대 할 수 없는 것

- **비율**: "20도는 10도의 2배"
- **곱하기/나누기**: 의미 있는 분석 불가능

### 🎓 시험에 나올 핵심

**Q. 등간척도의 가장 큰 특징은?**
- A) 간격이 일정하다 → **정답!**
- B) 비율 계산이 가능하다
- C) 절대영점이 있다
- D) 더하기가 불가능하다

**Q. 다음 중 등간척도는?**
1) 나이 - 아니오 (비율척도)

3) 키 - 아니오 (비율척도)
4) 학점 - 아니오 (서열척도)

## 📏 Level 4: 비율척도 (Ratio Scale) - "완벽한 숫자"

### 🔍 비율척도의 정의

**비율척도란**: 등간척도의 모든 특징에 **절대영점**까지 있는 가장 높은 수준의 척도입니다.

**쉽게 말하면**: "모든 연산이 완벽하게 가능한 진짜 숫자"입니다.

### 📌 비율척도의 특징

**1. 순서가 있다**
- 200cm > 150cm
- 명확한 크기 순서

**2. 간격이 일정하다**
- 150에서 170까지 = 20cm
- 170에서 190까지 = 20cm
- 간격 동일

**3. 절대영점이 있다** ⭐⭐⭐
- 0cm = 길이가 없음 (절대적으로!)
- 0kg = 무게가 없음 (절대적으로!)
- 0원 = 돈이 없음 (절대적으로!)

**4. 모든 연산이 가능하다**
- 더하기: 150cm + 50cm = 200cm ✓
- 빼기: 200cm - 150cm = 50cm ✓
- 곱하기: 100cm × 2 = 200cm ✓
- 나누기: 200cm ÷ 100cm = 2배 ✓

### 💡 비율척도의 실생활 예시

**키(높이)**
```
철수: 150cm
영희: 200cm
민준: 100cm

가능한 모든 분석:
- "영희는 철수보다 50cm 더 크다" ✓
- "영희의 키는 민준의 2배다" ✓ (비율 계산 가능!)
- 평균 키: (150 + 200 + 100) / 3 = 150cm ✓
- 표준편차: 계산 가능 ✓

왜 비율이 가능한가?
- 0cm는 "정말 키가 없음"을 의미
- 절대영점이 있기 때문
```

**몸무게**
```
A: 50kg
B: 100kg
C: 75kg

가능한 분석:
- "B는 A의 정확히 2배의 무게다" ✓
- 평균: 75kg ✓

왜 가능한가?
- 0kg = 정말로 무게가 없음
- 절대영점이 있음!
```

**나이(수명)**
```
할아버지: 80세
할머니: 75세
손자: 5세

가능한 분석:
- "할아버지는 손자의 16배 살았다" ✓
- 평균 나이: 53.3세 ✓

왜 가능한가?
- 0세 = 태어나지 않은 상태
- 절대영점이 명확
```

**소득(월급)**
```
직원 A: 2,000,000원
직원 B: 4,000,000원

가능한 분석:
- "B의 월급은 A의 정확히 2배다" ✓
- 평균 월급: 3,000,000원 ✓

왜 가능한가?
- 0원 = 정말로 돈이 없음
```

**시간(소요 시간)**
```
작업 A: 2시간
작업 B: 4시간

가능한 분석:
- "작업 B는 작업 A의 2배 시간이 걸린다" ✓

주의: "오전 10시"는 비율척도가 아님!
- 10시 × 2 = 20시 (의미 없음)
- 이건 절대영점이 없기 때문
```

### 📈 비율척도로 할 수 있는 분석

**1. 모든 것이 가능하다!**
- 명목척도: 빈도, 비율, 최빈값
- 서열척도: 중앙값, 순위 비교
- 등간척도: 평균, 표준편차
- 비율척도: **배수 비교, 기하평균, 변이계수**

**2. 배수 비교**
- "A는 B의 2배다"
- 비율 계산 가능

**3. 기하평균**
- 특별한 평균 계산 (성장률 같은 것)

**4. 변이계수 CV (Coefficient of Variation)**
- "표준편차 ÷ 평균 × 100"
- 서로 다른 크기의 데이터 비교할 때 사용
- **예**: A상품(평균 1000원) vs B상품(평균 10000원)의 변동성 비교

### 💻 변이계수 CV란 무엇인가?

**정의**: 평균에 비해 표준편차가 **얼마나 큰지**를 나타내는 지표

**공식**: CV = (표준편차 / 평균) × 100

**왜 필요한가?**
```
상품 A의 판매량
평균: 100개
표준편차: 10개

상품 B의 판매량
평균: 1000개
표준편차: 50개

누가 변동성이 더 클까?

표준편차만 보면:
- 상품 B의 50 > 상품 A의 10
- B가 더 크다?

변이계수로 보면:
- 상품 A: 10 / 100 × 100 = 10%
- 상품 B: 50 / 1000 × 100 = 5%
- A가 더 변동성이 크다!

결론: 상품 A가 더 불안정함!
```

**실생활 예시**
```
학생 A의 성적
평균: 90점
표준편차: 5점
CV = 5 / 90 × 100 = 5.6%

학생 B의 성적
평균: 70점
표준편차: 7점
CV = 7 / 70 × 100 = 10%

해석: 학생 B가 성적 변동이 더 크다
      (더 불안정한 성적)
```

### ❌ 비율척도에서 불가능한 것

**없다! 모든 연산이 가능하다!**

## 🎯 척도를 구분하는 팁

### Step 1: 순서가 있나?
- **없음** → 명목척도
- **있음** → 2단계로

### Step 2: 간격이 일정한가?
- **아니오** → 서열척도
- **예** → 3단계로

### Step 3: 절대영점이 있나?
- **없음** → 등간척도
- **있음** → 비율척도

### 실전 예제

**1. 영화 평점: 별 5개, 별 4개, 별 3개**
- 순서? 있음
- 간격? 불일정 (별 5개와 4개의 "품질 차이" ≠ 4개와 3개의 차이)
- → **서열척도**

**2. 온도: 10도, 20도, 30도**
- 순서? 있음
- 간격? 일정함 (매번 10도씩)
- 절대영점? 없음 (0도 = 온도 없음이 아님)
- → **등간척도**

**3. 키: 150cm, 170cm, 200cm**
- 순서? 있음
- 간격? 일정함
- 절대영점? 있음 (0cm = 길이 없음)
- → **비율척도**

**4. 지역: 서울, 경기, 인천**
- 순서? 없음
- → **명목척도**
