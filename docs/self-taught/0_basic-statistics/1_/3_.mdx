# 모집단과 표본: 전체를 알기 위한 부분의 과학

라면 국물의 간을 볼 때 냄비 전체를 다 마시지 않고 한 숟가락만 먹어보죠? 통계학도 이와 똑같습니다.

## 1. 핵심 용어 정리 필수 암기

| 구분 | 모집단 Population | 표본 Sample |
| :--- | :--- | :--- |
| **비유** | 국 냄비 전체 | 맛을 보는 한 숟가락 |
| **정의** | 관심의 대상이 되는 전체 집단 | 조사를 위해 선택된 일부 집단 |
| **특성치** | 모수 Parameter / 예: 모평균 μ, 모분산 σ2 | 통계량 Statistic / 예: 표본평균 x̄, 표본분산 s2 |
| **관계** | 우리가 알고 싶은 진짜 값 보통 모름 | 모수를 알기 위해 사용하는 도구 |
| **크기** | N 모두 포함 | n이고 n은 N보다 작음 |

### 모수와 통계량의 관계

- **모수**: 모집단의 특성을 나타내는 값 - 모평균 μ, 모표준편차 σ
- **통계량**: 표본의 특성을 나타내는 값 - 표본평균 x̄, 표본표준편차 s
- **목표**: 통계량을 이용해 모수를 추정

## 2. 조사 방법: 전수조사 vs 표본조사

### 전수조사 Census

**정의**: 모집단 전체를 조사하는 방법

**장점**
- 가장 정확한 결과 제공
- 표본오차가 없음

**단점**
- 막대한 비용 소요
- 매우 긴 조사 기간
- 실무적으로 불가능한 경우 많음

**사용 예시**
- 인구주택총조사 - 10년마다
- 정부 통계
- 의무적으로 필요한 경우

### 표본조사 Sample Survey

**정의**: 모집단 중 일부 표본만 뽑아서 조사하는 방법

**장점**
- 비용과 시간이 절약됨
- 실행 가능성 높음
- 빠른 결과 도출

**단점**
- 표본오차 발생
- 완전한 정확성 불가능
- 표본 구성에 따라 결과 달라질 수 있음

**사용 예시**
- 여론조사
- 선거 출구조사
- 상품 검사
- 의료 임상실험

### 표본오차와 신뢰도

표본조사에서는 반드시 표본오차가 발생합니다. 이를 관리하는 방법:
- **신뢰도 95%**: 같은 조사를 100번 하면 95번은 참값 근처
- **신뢰도 99%**: 더 정확하지만 표본 크기가 커짐
- **표본 크기**: 크수록 오차 감소 하지만 비용 증가

## 3. 표본추출 방법 4가지 시험 출제 1순위

표본은 모집단을 잘 대표해야 정확한 추론이 가능합니다. 이를 위해 랜덤 무작위 추출을 사용합니다.

### 1단순임의추출 Simple Random Sampling

**정의**: 제비뽑기처럼 모든 개체가 뽑힐 확률을 동일하게 하는 가장 기본적인 방법

**특징**
- 가장 공평함
- 편향 없음
- 모든 개체가 동일한 선택 확률

**방법**
- 난수표 사용
- 컴퓨터 랜덤 추출
- 제비뽑기

**예시**: 100명 중 10명을 뽑을 때, 모든 사람이 10 / 100 = 10% 확률로 선택됨

**장점**: 가장 순수한 무작위 추출
**단점**: 집단의 특성을 반영 못할 수 있음

### 2층화추출 Stratified Sampling - 비례 배분

**정의**: 모집단을 성격이 다른 여러 층 그룹으로 나누고, 각 층에서 골고루 뽑는 방법

**특징**
- 모집단의 다양성 반영
- 각 층의 특성 보존
- 높은 정확도

**방법**
1. 모집단을 여러 층으로 분류
2. 각 층의 비율 파악
3. 각 층에서 비례적으로 표본 추출

**예시**: 고등학교 학생 1,000명에서 100명을 뽑을 때
- 1학년: 300명 → 30명 선출
- 2학년: 350명 → 35명 선출
- 3학년: 350명 → 35명 선출

**장점**: 가장 정확도가 높음, 모집단 구조 반영
**단점**: 층의 구분이 명확해야 함

### 3집락추출 Cluster Sampling - 덩어리째

**정의**: 모집단을 여러 집락 덩어리으로 나누고, 몇 개 집락을 랜덤으로 뽑아 그 안을 다 조사하는 방법

**특징**
- 비용 효율적
- 광범위한 조사 용이
- 집락 내 개체는 모두 조사

**방법**
1. 모집단을 지역적 또는 자연적 집락으로 분류
2. 일부 집락을 랜덤 선택
3. 선택된 집락 내 모든 개체 조사

**예시**: 서울시 고등학교 학생 실태조사
- 서울시 650개 고등학교를 집락으로 분류
- 이 중 10개 학교를 랜덤 선택
- 선택된 10개 학교 전교생 조사

**장점**: 비용 절감 효과가 큼, 시간 효율적
**단점**: 정확도가 다소 낮음, 집락 간 차이 주의

### 4계통추출 Systematic Sampling - 규칙적

**정의**: 번호를 매기고 매 k번째 사람을 뽑는 방법

**특징**
- 간편한 실행
- 규칙적인 간격
- 실무에서 많이 사용

**방법**
1. 모집단에 번호 부여 - 1 부터 N까지
2. 표본 크기 n에 따라 추출간격 k = N / n 계산
3. 1부터 k까지 중 하나를 랜덤 선택 - 시작점 a
4. a, a + k, a + 2k, ... 선택

**예시**: 투표소에서 투표자 조사
- 투표자 1,000명 중 100명 조사
- 추출간격 k = 1,000 / 100 = 10
- 시작점을 4로 선택하면
- 4번, 14번, 24번, ... 1004번 투표자 조사

**장점**: 간편하고 빠름, 실행이 쉬움
**단점**: 주기적 패턴이 있으면 편향 발생 가능

## 4. 표본추출 방법 선택 기준

| 상황 | 추천 방법 | 이유 |
| :--- | :--- | :--- |
| 모집단이 동질적 | 단순임의추출 | 추가 분류 불필요 |
| 모집단이 이질적 | 층화추출 | 각 집단 특성 반영 가능 |
| 지역이 넓고 분산됨 | 집락추출 | 비용 절감 효과 |
| 명부가 있고 질서정연 | 계통추출 | 실행이 간단 |

## 5. 표본 크기 결정

**영향 요인**
- **신뢰도**: 높을수록 큰 표본 필요
- **오차 한계**: 작을수록 큰 표본 필요
- **모집단 크기**: 일반적으로 영향 적음 - 표본이 충분하면
- **비용**: 제한된 예산 내에서 결정

**일반적 기준**
- 표본 크기 30 이상: 대표성 있음
- 표본 크기 100-400: 일반적인 조사
- 표본 크기 1,000-2,000: 여론조사

## 6. 모집단과 표본의 중요성

정확한 추론을 위해서는:
- 모집단을 명확하게 정의
- 표본이 모집단을 잘 대표하도록 구성
- 적절한 표본추출 방법 선택
- 표본 크기 충분히 확보

이를 무시하면 편향된 biased 결과가 나와 잘못된 의사결정으로 이어질 수 있습니다.
