# 자료의 계층: 데이터의 종류를 제대로 알자

## 🎯 자료의 계층이란?

**쉽게 말하면**: 우리가 다루는 **데이터의 종류를 2가지로 먼저 나누고, 그 안에서 다시 세분화**하는 것입니다.

**왜 중요한가?**
- 데이터의 종류에 따라 **그릴 수 있는 그래프**가 다름
- 데이터의 종류에 따라 **사용할 수 있는 통계 분석**이 다름
- 잘못된 분류 = 잘못된 분석 = 잘못된 결론

:::info 핵심 질문
**"이 데이터로 평균을 낼 수 있는가?"**

이 질문으로 모든 게 결정됩니다!
:::

---

## 📊 자료의 첫 번째 분류: 질적 vs 양적

### 1-1. 질적 자료 (Qualitative Data) = 범주형 자료

**정의**: **"평균을 낼 수 없는"** 데이터입니다. 분류, 구분하는 것이 목적입니다.

**쉽게 말하면**: "이건 A 그룹, 이건 B 그룹"이라고 **분류하는 데이터**입니다.

#### 💡 질적 자료의 실생활 예시

**성별**
```
남 or 여
→ 성별의 평균? 말도 안 됨!
→ 할 수 있는 것: "남자 50명, 여자 40명이다" (개수 세기)
```

**선호하는 색깔**
```
빨강, 파랑, 노랑 중 선택
→ 색깔의 평균? 불가능!
→ 할 수 있는 것: "파란색을 좋아하는 사람이 40%이다"
```

**거주지**
```
서울, 경기, 인천, 대전...
→ 지역의 평균? 의미 없음!
→ 할 수 있는 것: "서울에 사는 학생이 300명이다"
```

**상품 카테고리**
```
의류, 식품, 전자제품
→ 카테고리의 평균? 불가능!
→ 할 수 있는 것: "의류 상품이 전체의 30%를 차지한다"
```

#### 📈 질적 자료로 할 수 있는 분석

- **빈도 Frequency**: "남자가 몇 명인가?" - 개수 세기
- **비율 Percentage**: "남자가 전체의 몇 %인가?" - 비율 계산
- **최빈값 Mode**: "가장 많은 것은?" - 가장 많이 나타나는 항목

#### ❌ 질적 자료로 절대 할 수 없는 것

- 평균: 성별의 평균? NO!
- 표준편차: 색깔의 표준편차? NO!
- 더하기/빼기: 남 + 여 = ? NO!

---

### 1-2. 양적 자료 (Quantitative Data) = 수치형 자료

**정의**: **"평균을 낼 수 있는"** 진짜 숫자 데이터입니다.

**쉽게 말하면**: "양이나 크기를 나타내는 **진짜 숫자**"입니다.

#### 💡 양적 자료의 특징

- **숫자로 표현**: 10, 20, 30, ...
- **연산 가능**: 더하기, 빼기 의미 있음
- **평균 계산 가능**: 의미 있는 분석
- **순서대로 정렬 가능**: 작은 것부터 큰 것 순서

#### 💡 양적 자료의 실생활 예시

**나이**
```
10세, 15세, 20세
→ 나이의 평균? 가능! (10 + 15 + 20) / 3 = 15세
```

**월급**
```
2,000,000원, 2,500,000원, 3,000,000원
→ 평균 월급? 가능! (2000 + 2500 + 3000) / 3 = 2,500,000원
```

**학생 수**
```
1반: 30명, 2반: 32명, 3반: 28명
→ 평균 학생 수? 가능! (30 + 32 + 28) / 3 = 30명
```

**시간**
```
1시간, 2시간, 3시간
→ 평균 시간? 가능! (1 + 2 + 3) / 3 = 2시간
```

## 📊 자료의 두 번째 분류: 이산형 vs 연속형

### ⚠️ 중요!
**이 분류는 양적 자료 안에서만 의미가 있습니다!**

**핵심 질문: "이 데이터를 셀 수 있는가, 잴 수 있는가?"**

---

## 🔢 2-1. 이산형 자료 (Discrete Data) - "셀 수 있는 것"

### 🔍 이산형 자료의 정의

**이산형이란**: **"하나, 둘, 셋"처럼 셀 수 있는** 데이터입니다.

**쉽게 말하면**: "개별 항목의 개수를 세는 데이터"입니다.

### 📌 이산형 자료의 특징

**1. 정수 값만 가능하다**
```
가능: 1개, 2개, 3개, ...
불가능: 1.5개, 2.3개
→ 아이는 1.5명일 수 없으니까!
```

**2. 소수점이 없다**
```
"자녀가 2명입니다" → 가능
"자녀가 1.8명입니다" → 불가능, 말도 안 됨!
```

**3. 값이 딱딱 끊어진다**
```
0, 1, 2, 3, 4, 5, ...
|   |   |   |   |   |
딱딱 끊어짐!

연속적이지 않고 띄엄띄엄!
```

### 💡 이산형 자료의 실생활 예시

**자녀 수**
```
가족 A: 0명
가족 B: 1명
가족 C: 2명
가족 D: 3명

왜 이산형인가?
- "자녀가 2.5명이다"? 불가능!
- 아이는 셀 수 있으니까
- 정수로만 표현 가능

분석:
- 평균 자녀 수: (0 + 1 + 2 + 3) / 4 = 1.5명 ✓
- 하지만 실제 누구도 1.5명을 낳지 않음!
```

**교통사고 건수**
```
월요일: 5건
화요일: 8건
수요일: 3건

왜 이산형인가?
- "사고가 4.5건이다"? 말도 안 됨!
- 사고는 일어나거나 안 일어나거나!
- 0, 1, 2, 3, ... 개수로만 표현
```

**책의 페이지 수**
```
책 A: 200페이지
책 B: 350페이지
책 C: 280페이지

왜 이산형인가?
- "페이지가 250.5페이지다"? 불가능!
- 페이지는 세므로 정수로만!
```

**주사위 눈**
```
1, 2, 3, 4, 5, 6
→ "3.5가 나왔다"? 불가능!
→ 주사위는 정수값만 가능
```

**결석한 학생 수**
```
1월: 5명
2월: 8명
3월: 2명

왜 이산형인가?
- "3.2명이 결석했다"? 불가능!
- 학생은 셀 수 있으니까
```

### 📈 이산형 자료로 할 수 있는 분석

- **평균**: (1 + 2 + 3) / 3 = 2 ✓
- **표준편차**: 데이터가 얼마나 흩어져 있는가
- **중앙값**: 중간값
- **더하기**: 의미 있음
- **그래프**: 기둥 그래프, 히스토그램

### ❌ 이산형 자료의 주의점

```
평균이 1.8명이라고 해서
"일반적으로 1.8명을 낳는다"는 뜻이 아님!

실제로는:
- 어떤 가정은 0명
- 어떤 가정은 1명
- 어떤 가정은 3명
평균이 1.8일 뿐!
```

## 📏 2-2. 연속형 자료 (Continuous Data) - "잴 수 있는 것"

### 🔍 연속형 자료의 정의

**연속형이란**: **자로 재는 것처럼** 값이 연속적으로 이어지는 데이터입니다.

**쉽게 말하면**: "소수점으로 무한히 세분화할 수 있는 데이터"입니다.

### 📌 연속형 자료의 특징

**1. 소수점이 가능하다**
```
170cm, 170.5cm, 170.51cm, 170.512cm, ...
무한히 세밀하게 표현 가능!
```

**2. 값이 연속적이다**
```
150 ←→ 160 ←→ 170 ←→ 180
그 사이에도 무한히 많은 값!

150.1, 150.2, 150.25, 150.251, ...
계속 세분화 가능!
```

**3. 정밀도에 따라 표현한다**
```
일반 자: 170cm
더 정밀한 자: 170.5cm
더더 정밀한 자: 170.51cm

모두 맞는 표현!
정밀도만 다를 뿐
```

### 💡 연속형 자료의 실생활 예시

**키**
```
철수: 170cm
영희: 170.5cm
민준: 170.25cm

왜 연속형인가?
- 자로 재서 표현하는 데이터
- "170.7cm"라고 할 수 있음
- 소수점이 의미 있음

분석:
- 평균 키: (170 + 170.5 + 170.25) / 3 = 170.25cm ✓
- "평균 키가 170.25cm"는 의미 있음!
```

**몸무게**
```
A: 65kg
B: 65.3kg
C: 65.31kg
D: 65.312kg

왜 연속형인가?
- 저울의 정밀도에 따라 소수점 결정
- 이론상 무한히 세분화 가능
- 5.5kg 가능? 네!
```

**온도**
```
아침: 10℃
점심: 20.5℃
저녁: 15.3℃

왜 연속형인가?
- 온도계의 정밀도에 따라 다름
- "20.25℃"라고 할 수 있음
- 소수점이 의미 있음
```

**강수량**
```
월요일: 10mm
화요일: 10.5mm
수요일: 10.23mm

왜 연속형인가?
- 강수계로 재는 데이터
- "10.251mm"이라고 표현 가능
```

**소비 시간**
```
공부 시간:
학생 A: 3시간
학생 B: 3.5시간
학생 C: 3.25시간

왜 연속형인가?
- 시간은 연속적으로 흘러감
- "3.33시간"이라고 할 수 있음
```

**소득**
```
A씨: 2,000,000원
B씨: 2,500,000원
C씨: 2,350,000원

연속형일까?
- 이론상으로는 연속형
- 돈은 1원 단위로 쪼갤 수 있으니까
- "2,500,000.5원"은 실제로는 불가능하지만
- 통계학적으로는 연속형으로 취급
```

### 📈 연속형 자료로 할 수 있는 분석

- **평균**: 의미 있음
- **표준편차**: 의미 있음
- **중앙값**: 의미 있음
- **그래프**: 히스토그램, 산점도, 상자 그림
- **모든 통계 분석** 가능

## 🆚 이산형 vs 연속형: 핵심 차이

| 항목 | 이산형 | 연속형 |
| :--- | :---: | :---: |
| **정의** | 셀 수 있음 | 잴 수 있음 |
| **예시** | 1, 2, 3 | 1.1, 1.11, 1.111 |
| **소수점** | 없음 | 있음 |
| **표현** | 정수만 | 무한히 세밀 |
| **공통점** | 모두 양적 자료 - 평균 계산 가능! |
| **실생활** | 자녀 수, 사고 건수 | 키, 온도, 시간 |

### ⚠️ 혼동하기 쉬운 경우!

**나이: 이산형인가, 연속형인가?**
```
일상에서 표현: "25살입니다" (정수, 이산형?)
통계학에서: 실제로는 25.7살일 수 있으니 연속형!
→ 단순히 반올림해서 정수로 표현할 뿐!

결론: 연속형으로 봐야 함
```

**금액: 이산형인가, 연속형인가?**
```
"2,500,000원" → 이산형! (원 단위)
→ 2,500,000.5원? 불가능!
→ 가장 작은 단위가 1원이므로

결론: 이산형
```

**신발 사이즈: 이산형인가, 연속형인가?**
```
210, 220, 230, 240, 250, ...
→ 215? 225? 불가능!
→ 정해진 사이즈만 존재

결론: 이산형 (정해진 값만)
```

## 📊 자료의 계층 전체 구조

```
자료
│
├─ 질적 자료 (평균 불가능)
│  ├─ 명목형: 성별, 종교, 지역
│  └─ 순서형: 학점, 만족도, 등급
│
└─ 양적 자료 (평균 가능)
   ├─ 이산형: 자녀 수, 사고 건수, 페이지 수
   └─ 연속형: 키, 온도, 시간, 무게
```

### 🎯 핵심 비교표

| 종류 | 질적자료 명목형 | 질적자료 순서형 | 양적자료 이산형 | 양적자료 연속형 |
| :--- | :---: | :---: | :---: | :---: |
| **평균** | ✗ | ✗ | ✓ | ✓ |
| **순서** | ✗ | ✓ | ✓ | ✓ |
| **소수점** | - | - | ✗ | ✓ |
| **예시** | 성별 | 학점 | 자녀 수 | 키 |

---

## 🎯 자료를 구분하는 방법

### Step 1: 평균을 낼 수 있는가?

```
평균 가능?
├─ 아니오 → 질적 자료
│         (명목형 또는 순서형)
└─ 예 → 양적 자료로 진행
```

### Step 2: 순서가 있는가?

```
(질적 자료일 때)
순서 있나?
├─ 아니오 → 명목형
│         (성별, 종교 등)
└─ 예 → 순서형
      (학점, 등급 등)
```

### Step 3: 셀 수 있는가, 잴 수 있는가?

```
(양적 자료일 때)
셀 수 있나?
├─ 예 (정수 1, 2, 3...) → 이산형
│                        (자녀 수 등)
└─ 아니오 (소수 1.1, 1.5...) → 연속형
                              (키 등)
```

## 📚 실전 예제로 완벽히 이해하기

### 예제 1: "각 학생의 영어 성적"

```
학생 A: 90점
학생 B: 85점
학생 C: 95점

→ 평균 가능? 예!
→ 양적 자료

→ 셀 수 있나? 
   음... 이론상 89.5점도 가능
   (온라인 시험은 소수점 반영)
→ 연속형

결론: 연속형 자료
분석 가능: 평균, 표준편차, 그래프 등 모두 가능!
```

### 예제 2: "만족도 조사 (매우만족, 만족, 보통, 불만족)"

```
→ 평균 가능? 이론상 가능하지만... 의미가 있나?
→ 질적 자료

→ 순서 있나? 예!
   (매우만족 > 만족 > 보통 > 불만족)
→ 순서형

결론: 순서형 자료 (질적)
분석 가능: 최빈값, 중앙값은 가능
          하지만 평균은 통계적으로 부정확할 수 있음
```

### 예제 3: "월급 (A씨: 2,500,000원, B씨: 3,000,000원)"

```
→ 평균 가능? 예! (2,500,000 + 3,000,000) / 2 = 2,750,000원
→ 양적 자료

→ 셀 수 있나? 
   1원 단위까지만 가능 (2,500,000.5원은 불가능)
→ 이산형

결론: 이산형 자료
분석 가능: 평균, 표준편차 등 모두 가능!
```

### 예제 4: "거주지 (서울, 경기, 인천, 대전)"

```
→ 평균 가능? 아니오!
→ 질적 자료

→ 순서 있나? 아니오!
→ 명목형

결론: 명목형 자료
분석 가능: 빈도, 비율만 가능
          (서울에 사는 학생이 40%)
```

## 💡 자료의 계층이 중요한 이유

### 그릴 수 있는 그래프가 달라진다

```
명목형: 막대그래프, 원그래프
순서형: 막대그래프 (순서 유지)
이산형: 기둥 그래프, 히스토그램
연속형: 히스토그램, 산점도, 상자 그림
```

### 사용할 수 있는 통계가 달라진다

```
명목형: 빈도, 비율, 최빈값만!
순서형: 위 + 중앙값
이산형: 위 + 평균, 표준편차 (모두!)
연속형: 모든 통계 분석 가능 (가장 강력함!)
```

### 잘못된 분석의 위험성

```
예: 학점 A, B, C의 평균을 낸다
평균: (4.0 + 3.0 + 2.0) / 3 = 3.0

하지만 이건 문제가 있다!
- A와 B의 차이 ≠ B와 C의 차이
- 간격이 일정하지 않으니까!
- 통계적으로 부정확한 분석

→ 자료의 계층을 알면 이런 실수를 막을 수 있음!
```
