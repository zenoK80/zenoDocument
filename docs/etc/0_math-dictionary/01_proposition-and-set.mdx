# 명제와 집합

명제와 집합은 수학의 가장 기본이 되는 개념입니다. 이 단원에서는 명제의 참과 거짓을 판단하고, 집합의 개념과 연산을 학습합니다.

## 핵심 개념

명제 p가 q를 함의한다는 것(p ⇒ q)은 집합으로 표현하면 P ⊂ Q와 같습니다. 즉, 명제와 집합은 서로 밀접한 관계를 가지고 있습니다.

---

## 1. 명제란 무엇인가?

### 명제의 정의

**명제(proposition)**는 참(true)이거나 거짓(false)인지를 명확하게 판단할 수 있는 문장이나 식을 의미합니다.

예를 들어:
- "2 + 3 = 5"는 참인 명제입니다.
- "서울은 대한민국의 수도이다"는 참인 명제입니다.
- "x = 1"은 x의 값에 따라 참 또는 거짓이 결정되므로, x가 주어지지 않으면 명제가 아닙니다.

### 명제가 아닌 것들

다음과 같은 문장들은 명제가 아닙니다:
- "얼마나 아름다운가!" (감탄문)
- "창문을 닫아라" (명령문)
- "오늘 날씨가 좋니?" (의문문)

이러한 문장들은 참, 거짓을 판단할 수 없거나 의미가 명확하지 않기 때문입니다.

### 조건을 포함한 명제

"x = 1이면 x² = 1"과 같은 형태의 명제를 **조건 명제**라고 합니다. 여기서:
- x = 1은 **조건 p**
- x² = 1은 **결론 q**

이를 p ⇒ q로 표기하며, "p이면 q이다" 또는 "p는 q의 충분조건이다"라고 읽습니다.

---

## 2. 집합의 기본 개념

### 집합이란?

**집합(set)**은 특정한 조건을 만족하는 대상들의 모임입니다. 집합을 이루는 각각의 대상을 **원소**(element)라고 합니다.

집합의 표현 방법:
1. **원소 나열법**: A = \{1, 2, 3, 4, ...\} 처럼 원소를 직접 나열합니다.
2. **조건 제시법**: A = \{x | x > 0, x는 정수\} 처럼 조건으로 표현합니다.

여기서 세로 막대 | 는 "~인 x"를 의미하며, "such that"으로 읽습니다.

### 집합의 표기

P = \{x | p(x)\}로 표기할 때:
- x는 집합의 구성원을 나타내는 변수
- p(x)는 x가 만족해야 할 조건

예시: "양수의 집합"은 A = \{x | x > 0\}로 표현할 수 있습니다.

---

## 3. 명제와 집합의 관계

### p ⇒ q와 P ⊂ Q

두 명제 p와 q에 대해 "p이면 q"(p ⇒ q)가 참이라는 것은:
- 조건 p를 만족하면 반드시 조건 q를 만족한다는 의미입니다.
- 이는 집합으로 표현하면 **P ⊂ Q**(P는 Q의 부분집합)와 동일합니다.

예를 들어, 명제 "x = 1이면 x² = 1"이 참이라면:
- P = \{x | x = 1\}
- Q = \{x | x² = 1\}
- 이때 P ⊂ Q가 성립합니다.

### 쌍방향 함의와 집합의 동등

"p ⇒ q이고 q ⇒ p"일 때, 이를 "p ⇔ q"로 표기하며 "p와 q는 동치"라고 합니다.

집합으로 표현하면:
- p ⇒ q 그리고 q ⇒ p이므로
- P ⊂ Q 그리고 Q ⊂ P
- 따라서 P = Q

이는 두 집합이 완전히 같다는 것을 의미합니다.

### 벤 다이어그램으로 이해하기

전체 집합(전집)을 직사각형으로, 집합 P와 Q를 원이나 타원으로 표현하면:

1. **p ⇒ q인 경우**: P가 Q 안에 포함된 형태
2. **p ⇔ q인 경우**: P와 Q가 완전히 겹치는 형태

---

## 4. 실전 예제

### 예제: 부등식과 집합

다음 명제를 검증해봅시다:

**"−1 < x < 1 ⇒ x < 2"가 참인지 거짓인지 조사하시오.**

**[풀이]**

−1 < x < 1을 만족하는 집합을 P, x < 2를 만족하는 집합을 Q라고 하면:

- P는 −1과 1 사이의 모든 수
- Q는 2보다 작은 모든 수

수직선 위에 표시하면:
```
    -2      -1       0       1       2
    |-------|--------|-------|-------|
              [-------P-------]
    [-----------Q----------------]
```

P의 모든 원소가 Q에 포함되므로 P ⊂ Q입니다.

**따라서 명제 "−1 < x < 1 ⇒ x < 2"는 참입니다.**

## 학습 포인트

이 단원을 통해 다음을 이해해야 합니다:

- 명제는 참 또는 거짓을 명확히 판단할 수 있는 문장이나 식입니다.
- 집합은 특정 조건을 만족하는 원소들의 모임입니다.
- 명제의 함의 관계(p ⇒ q)는 집합의 포함 관계(P ⊂ Q)와 대응됩니다.
- 벤 다이어그램을 활용하면 명제와 집합의 관계를 시각적으로 이해할 수 있습니다.

명제와 집합의 관계를 이해하면, 복잡한 수학적 논리를 더 명확하게 파악할 수 있습니다.
